# MedRAG-X

MedRAG-X is an AI-driven medical imaging platform that performs
automatic brain tumor segmentation using 3D U-Net and enables
semantic medical knowledge retrieval using vector databases.

## Features
- 3D MRI tumor segmentation
- BraTS dataset ingestion
- MONAI-based preprocessing
- Deep learning training pipeline
- Multimodal embeddings
- Semantic search
- RAG-ready architecture

## Dataset
BraTS 2020 MRI dataset

## Model
- 3D U-Net
- Dice Loss
- Multi-class segmentation

## Future Scope
- Clinical LLM integration
- Radiology report generation
- Explainable AI (GradCAM)
- PACS integration
